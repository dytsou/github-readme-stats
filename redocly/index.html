GitHub Readme Stats API (1.0.2)

Download OpenAPI specification:

URL: https://github.com/dytsou/github-readme-stats License: MIT

Dynamically generated GitHub stats cards for your README.

This API provides endpoints to generate SVG cards displaying GitHub statistics, top languages, repository information, Gists, and WakaTime statistics.

All card endpoints return SVG images that can be embedded in README files.

Cards

SVG card generation endpoints
